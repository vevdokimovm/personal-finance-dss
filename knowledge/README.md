# knowledge/ — база знаний проекта FINPILOT

Не-кодовая база знаний. Живёт только в приватном репозитории (`personal-finance-dss`),
в публичное зеркало `finpilot` не публикуется.

## Разделы
- **business/** — стратегия, тактика, анализ рынка, монетизация, философия продукта,
  go-to-market (Google Play / App Store пайплайны), операционный плейбук, реестр постов.
- **research/** — опросы, анализ аудитории, user-портреты, экспертные оценки, методички
  по анализу исследований.
- **legal/** — юридический разбор (152-ФЗ и пр.), шаблоны юридических документов.
- **guides/** — переиспользуемые методички, руководства и справочники (не привязаны к
  коду FINPILOT). `guides/templates/` — чистые шаблоны артефактов (SRS, Requirements SOP).
- **product/** — пользовательская документация продукта и инструкции запуска.
- **brand/** — логотип, паспорт логотипа, визуальные ассеты.
- **project_meta/** — исторические инструкции проекта по аккаунтам.
