## Продукт
FINPILOT — прескриптивная СППР по личным финансам: рекомендует, как распределить
свободный денежный поток между досрочным погашением долга, резервом ликвидности и
финансовыми целями. Ключевой дифференциатор — прозрачность и объяснимость
(human-readable объяснения, не «чёрный ящик»). Цель — коммерческий продукт уровня
Т-Банк/Альфа/Сбер, публичный запуск к концу лета 2026. ВКР защищена: академическую
рамку (ГОСТ, «дипломный проект») не поднимать — это продакшн-продукт.

## Стек
Python/FastAPI, PostgreSQL (prod) / SQLite (test), SQLAlchemy 2.0, Pydantic v2,
Jinja2 + vanilla JS, JWT-аутентификация.

## Канон математической модели — ИСТОЧНИК ИСТИНЫ КОД, не доки
- Альтернативы: stars-and-bars, шаг 10% → 66 комбинаций (app/core/alternatives.py).
  Старая спецификация «21 альтернатива / шаг 20% / flow-based Lt / MC 95%» — УСТАРЕЛА,
  игнорировать полностью.
- Ликвидность Lt: stock-based, в месяцах автономии; L_MIN = 0.0 (по ликвидности не
  отсеиваем); ПДН dt_max = 0.40 (app/core/filtering.py).
- Прогноз: SES α=0.3, Monte-Carlo N=1000, доверительный интервал 80% [p10..p90]
  (app/core/forecast.py).
- 5 профилей риска (1 Консервативный … 5 Агрессивный; по умолчанию 3 Сбалансированный);
  веса (w_rt, w_lt, w_dt, w_goals) — в app/core/ranking.py.
- Avalanche: досрочка только по обязательствам со ставкой ≥ r_bench (app/core/avalanche.py).
- Полная спецификация — docs/Математическая_модель_v3_0_0.md. При расхождении доки с
  кодом прав КОД.

## Жёсткие правила работы (не дрейфовать)
- TDD обязателен: red → green, тесты прогонять реально, зелёное не фейковать.
- SemVer: фикс = PATCH, фича = MINOR, ломающее = MAJOR.
- НИКАКОГО массового реформата: `black .` запрещён (ломает merge между двумя
  аккаунтами Claude, на которых ведётся работа).
- Строгий порядок роадмапа P0 → P1 → P2 → P3, без самовольной смены приоритетов.
- WATCHLOG (docs/WATCHLOG.md) — вахтенный журнал, ведётся непрерывно по ходу работы;
  в конце батча отдаётся ОТДЕЛЬНЫМ файлом в чат + идентичная копия внутри архива.
- Архив на батч: одна папка внутри, имя finpilot_vX_Y_Z_intl.
- «продолжай» = действуй без ожидания подтверждения.
- Код — источник истины: не верить устаревшим audit-докам, читать реальные файлы,
  гонять тесты.
- Грабли песочницы Claude — в FINPILOT_Claude_sandbox_runbook.md, заново не
  диагностировать (dash-shell, вывод в файл вместо pipe, setsid для долгих прогонов,
  chromium --no-sandbox, E2E запускаются через `-m e2e`).
