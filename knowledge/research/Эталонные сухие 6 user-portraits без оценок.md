# Эталонные пользовательские профили FINPILOT — входные данные

> **Справочник для тестирования.** Содержит исходные данные шести типовых пользовательских профилей: демографию, доходы, расходы, обязательства, цели и ликвидную позицию. Без вычислений показателей, без оценок алгоритма, без интерпретаций — только входные данные, которые подаются в модель.

---

## 0. Структура каждого профиля

1. **Демография** — имя, возраст, профессия, город, семейное положение, краткий контекст.
2. **Доходы** — суммарный месячный доход и его структура.
3. **Расходы** — суммарные жизненные траты в месяц (без обязательных платежей по кредитам).
4. **Обязательства** — таблица кредитов: банк, тип, остаток долга, ежемесячный платёж, ставка, оставшийся срок.
5. **Ликвидная позиция** — текущий баланс на счетах + отдельные накопительные инструменты (депозиты).
6. **Цели накопления** — таблица: название, целевая сумма, накоплено, осталось, дедлайн, процент готовности.

Все суммы — в рублях.
