# Глоссарий показателей и терминов FINPILOT

Единый словарь: каждый показатель определён один раз и однозначно. Используется как источник
истины для UI-копирайта, онбординга и защиты.

---

## Базовые показатели

| Термин | Определение | Формула | Норма |
|---|---|---|---|
| **CFt** — чистый денежный поток | Сколько остаётся после жизненных расходов, до выплат по кредитам | `I − E` | `> 0` |
| **Rt** — свободные деньги (доступный ресурс) | Остаток после всех обязательных платежей: жизни и кредитов. Именно он распределяется | `CFt − ΣP` | `> 0` |
| **Lt** — функциональная ликвидность | Доля от обязательных платежей, которую можно направить в накопления/досрочку. Показатель оперативной устойчивости месяц-к-месяцу | `Rt / (E + ΣP)` | `≥ 0.30` |
| **Dt** — долговая нагрузка (ПДН) | Доля доходов на обязательные кредитные платежи | `ΣP / I` | `≤ 0.40` |
| **BLR** — базовый запас ликвидности | **Сколько месяцев** можно прожить без дохода, тратя наличные и ликвидную позицию | `(Bt + Bliq) / E` | `2.5 … 6 мес` |
| **Sn** — обеспеченность целей | Какая доля целей уже накоплена (взвешенно по категориям и срочности) | см. модель §11 | контекстно |

---

## ⚠️ Lt и BLR — это РАЗНЫЕ вещи

Частый источник путаницы. Оба про «ликвидность», но измеряют разное и в разных единицах:

- **Lt — функциональная ликвидность**, безразмерная доля. Отвечает на вопрос: *«какую часть
  обязательных платежей я могу дополнительно куда-то направить прямо сейчас»*. Формула `Rt / (E + ΣP)`.
- **BLR — запас ликвидности в месяцах**. Отвечает на вопрос: *«сколько месяцев я протяну без дохода»*.
  Формула `(Bt + Bliq) / E`.

«Подушка на 3 месяца» — это **BLR ≥ 3**, а **не** `Lt ≥ 3`. Порог `Lt ≥ 0.30` — про функциональную
ликвидность, и трактовать его как «3 месяца» неверно.

## Алгоритм и решение

| Термин | Значение |
|---|---|
| **a\*** | Оптимальное распределение свободных денег — тройка (досрочка, резерв, цели) |
| **U(a)** | Функция полезности альтернативы (SAW — взвешенная сумма нормированных критериев), `∈ [0, 1]` |
| **Профиль риска** | Один из пяти наборов весов критериев: консервативный → агрессивный |
| **Avalanche** | Стратегия досрочки: гасим кредит с наивысшей ставкой первым |
| **r_bench (OCR)** | Пороговая ставка: досрочка выгодна только для кредитов со ставкой выше альтернативной доходности (по умолчанию 0.14) |
| **MC доверительный интервал** | Диапазон прогноза по Monte-Carlo. В реализации — **80% [p10..p90]** (см. `docs/model_vs_code.md`) |
| **fail-loud** | Принцип: при `Rt ≤ 0` или дефиците алгоритм отказывается советовать и сообщает о проблеме, а не выдаёт пустую/выдуманную рекомендацию |
