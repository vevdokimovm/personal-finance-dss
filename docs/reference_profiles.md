# Эталонные профили (ресинхронизация через движок)

Шесть эталонных профилей, **пересчитанных реальным движком FINPILOT** (v2.1.0).
Значения получены вызовом `/api/planning/calculate` на демо-данных (профиль риска по умолчанию —
сбалансированный, `r_bench = 0.14`). Это канонический эталон: ожидаемые `a*` по определению
совпадают с выходом движка и должны использоваться для валидации вместо устаревших расчётов.

> Демо-кейсы загружаются в приложении (боковая панель) или через `POST /api/demo/load?case=<имя>`.

---

## Сводная таблица

Обозначения: **I** — доход, **E** — расходы, **ΣP** — платежи по кредитам, **Rt** — свободные деньги,
**Lt** — функциональная ликвидность, **Dt** — долговая нагрузка (ПДН), **BLR** — месяцы жизни без дохода.
**a\*** — рекомендованное распределение (досрочка / резерв / цели).

## Пояснения к ключевым случаям

- **mikhail — отказ (fail-loud).** При Rt = −3 000 ₽ свободных денег нет: расходы и платежи
  превышают доход. Движок **не выдаёт рекомендацию** (0 допустимых альтернатив), а сигнализирует
  о структурной проблеме. Это поведение по принципу fail-loud, а не сбой.
- **olga — досрочка.** Единственный профиль, где оптимум — направить всё на досрочное погашение
  (ставка кредита выше порога `r_bench = 0.14`, досрочка экономически выгоднее накоплений).
- **anna / dmitriy / igor / viktor — цели.** При здоровых показателях и кредитах дешевле бенчмарка
  (или их отсутствии) оптимум — финансирование целей. Дешёвый кредит не проходит OCR-фильтр Avalanche,
  поэтому сумма «пересыпается» из досрочки в цели.
- **igor / viktor — 6 альтернатив.** Без кредитов размерность задачи падает: переменная досрочки
  вырождается, остаётся распределение между резервом и целями.

## Как воспроизвести

```bash
# на каждом кейсе
curl -X POST "http://localhost:8000/api/demo/load?case=anna"
curl -X POST "http://localhost:8000/api/planning/calculate" -H "Content-Type: application/json" -d '{}'
```

Поле `best` ответа — рекомендованное `a*`; `indicators` — базовые показатели; `admissible_count` —
число прошедших фильтр альтернатив.
