# FINPILOT — Диаграммы для ВКР

Все 20 диаграмм продукта **FINPILOT** (СППР в персональных финансах) в формате **draw.io** с ручной раскладкой — открываются чисто, без наложений стрелок и подписей.

## Как открыть и вставить в ВКР

1. Зайди на **app.diagrams.net** (это и есть draw.io, бесплатно, без регистрации)
2. **File → Open From → Device** → выбери `.drawio` файл
3. Для вставки в Word/PDF: **File → Export as → PNG** (поставь Zoom 200–300% для качества) или **PDF**
4. Можно открыть локально в десктоп-приложении draw.io или через плагин в VS Code / PyCharm

## Главный файл

**`FINPILOT_все_диаграммы.drawio`** — все 20 диаграмм в одном файле, переключаются вкладками снизу. Удобно листать на защите. Отдельные файлы тоже лежат рядом, если нужно экспортировать по одной.

## Единая цветовая схема

Во всех диаграммах слои продукта окрашены одинаково, чтобы читалось консистентно:
- 🔵 **синий** — слой API (`app/api`)
- 🟢 **зелёный** — сервисы (`app/services`)
- 🟡 **жёлтый** — алгоритмическое ядро (`app/core`)
- 🔴 **красный** — база данных (`app/database`)

ГОСТ-блок-схемы используют стандартные фигуры: овал-терминатор (начало/конец), параллелограмм (ввод/вывод), ромб (решение), прямоугольник (процесс), прямоугольник с боковыми линиями (предопределённый процесс), шестиугольник (подготовка).
