# Реестр инцидентов FINPILOT

Краткие сводки по каждому инциденту — для быстрой ориентации разработчика/AI в новой сессии.
**Полные разборы (post-mortem)** живут ОТДЕЛЬНЫМИ файлами вне репозитория (у владельца проекта +
в Claude-проекте), оформляются по правилу §12 `WATCHLOG.md`. Здесь — только сводка: что было,
корень, как закрыто. Формат самих разборов — `docs/incident_postmortem.md`.
