Согласие на обработку финансовых данных

Сервис FINPILOT · редакция от {{ legal.doc_date }}

Финансовые сведения чувствительны, поэтому их обработка оформляется отдельным явным согласием. Сервис обрабатывает финансовые данные исключительно для формирования персональных показателей и рекомендаций и не проводит операций по счетам.

Подтверждая настоящее согласие, я поручаю сервису FINPILOT обработку финансовых данных, предоставленных мной одним из следующих способов:

  • путём ручного ввода данных в интерфейсе Сервиса;
  • путём загрузки файла банковской выписки;
  • путём передачи мной токена доступа к API счёта, по которому я являюсь правомочным владельцем.

1. Состав данных

История операций, суммы, даты, назначения и категории платежей, остатки по счетам.

2. Цель обработки

Исключительно формирование персональных финансовых показателей и рекомендаций внутри Сервиса.

3. Подтверждения Пользователя

  • предоставляемые данные относятся к моим собственным счетам;
  • я вправе передавать эти данные Сервису;
  • Сервис не инициирует операций по списанию средств и не имеет к ним доступа.

4. Передача третьим лицам

Данные не передаются третьим лицам, за исключением случаев, прямо предусмотренных законом.

5. Отзыв

Я могу в любой момент отозвать согласие и удалить импортированные данные через интерфейс Сервиса либо обратившись на {{ legal.contact_email }}.
